# Canvas Generator

A simple tool to transform ideas into structured features using two canvases:

1. **Value First Canvas** - Data → Solution → Insights → Actions → Value
2. **AI Orchestration Framework Canvas** - Predictive, Generative, Agentic AI planning

Inspired by [Xebia's AI For Product Management](https://xebia.com) frameworks.

## Usage

1. Visit [Canvas Generator](https://gerritvos.github.io/canvasgenerator/)
2. Fill in your idea and canvas fields
3. Click "Generate Output" to preview
4. Download as Markdown or PDF

No API keys required - works entirely in your browser.
